# Searchbar_App

# React JS

- [React](https://facebook.github.io/react/) and [React Router](https://reacttraining.com/react-router/) for the frontend


## Requirements

- [Node.js](https://nodejs.org/en/) 6+

```shell
npm install
```


### Running

Runs the app in the production mode using a node server.<br>

Production mode:

```shell
npm start
```
